# CropMonitor

국외작황 모니터링 활용시스템

## 프로젝트 소개

전 세계 주요 농작물의 작황 정보를 모니터링하고 시각화하는 웹 애플리케이션입니다.

## 현재 구현된 페이지

### 1. 랜딩 페이지 (index.html)
- 메인 로고 및 타이틀 표시
- 서비스 소개
- 지도 페이지로 이동하는 시작하기 버튼
- 참여 기관 로고 표시
- 전북대학교 스마트팜학과 정보가 포함된 푸터

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript

## 실행 방법

1. 해당 프로젝트를 git clone을 통해 복제
2. vscode를 통해 실행
3. live-server를 통해 실행 후 `127.0.0.1:5500`으로 접속
4. 실행된 페이지를 확인
